About

Gateway Hackers is a news link site for St. Louisans who make stuff with computers.

Follow @gatewayhackers

Contact

Please direct questions and comments to contact@gatewayhackers.com.

Link Submission Guidelines

Basically, if a programmer, designer, tester, team lead, or anyone involved in IT or running a startup might find it interesting and it doesn't violate the Code of Conduct, post it. We're especially interested in links focused on the greater St. Louis area.

Code of Conduct

Gateway Hackers strives to be an inclusive, diverse and welcoming place. Our community guidelines are based on the Speak Up! Code of Conduct:

  • Be welcoming, friendly, and patient.
  • Be considerate.
  • Be respectful.
  • Be careful in the words that you choose.
  • When disagreeing, try to understand why.

Behind the Scenes

Gateway Hackers started as a personal project in October 2013 by Mike Janger, a St. Louis developer. Inspired by Hacker News, Reddit, Lobsters, and a few other sites, Gateway Hackers provides a place for St. Louis residents to share and discuss tech news. The site is powered by Linode, Ubuntu, nginx, the Python programming language and the Django web framework. Source is available on Github.
